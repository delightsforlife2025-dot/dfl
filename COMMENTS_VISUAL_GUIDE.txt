═══════════════════════════════════════════════════════════════════════════════
                      INTERACTION FLOWS
═══════════════════════════════════════════════════════════════════════════════

APPROVE A COMMENT:
──────────────────
1. Go to /dashboard/comments
2. Click comment in list
3. In right panel, click "[✓ Onayla]" button
4. Status changes to "✓ Onaylı"
5. Comment appears on homepage
6. Done! ✓


REJECT A COMMENT:
─────────────────
1. Go to /dashboard/comments
2. Click comment in list
3. Status shows "✓ Onaylı"
4. Click "[✓ Onayla Kaldır]" button
5. Status changes to "Beklemede"
6. Comment hidden from homepage
7. Done! ✓


DELETE A COMMENT:
────────────────
1. Go to /dashboard/comments
2. Click comment in list
3. Click "[✗ Sil]" button
4. Confirmation dialog appears:
   "Bu yorumu silmek istediğinizden emin misiniz?"
5. Click "Evet, Sil"
6. Comment removed from list
7. Done! ✓


FILTER COMMENTS:
────────────────
1. Go to /dashboard/comments
2. Click filter tab:
   - "Tüm Yorumlar" → Shows all
   - "Onaylı" → Shows approved only
   - "Beklemede" → Shows pending only
3. List updates instantly
4. Done! ✓


ADD NEW COMMENT:
────────────────
1. Go to /dashboard/comments
2. Click "[+ Yorum Ekle]" button
3. Fill form fields
4. Check "yayınla (onayla)" if approving
5. Click "Yorum Ekle"
6. See success message
7. Auto-redirect to comments list
8. New comment appears in list
9. Done! ✓

COMMENT STATUS BADGES:
──────────────────────
✓ Onaylı   → Green badge (published, shows on homepage)
⏳ Beklemede → Yellow badge (pending, hidden from public)


LOADING STATES:
───────────────
• Skeleton screens while fetching
• Spinner icon during submission
• Disabled buttons while loading


ERROR STATES:
──────────────
⚠️ Red error message box with icon
• Shows validation errors
• Shows submission errors
• Clear error messages


SUCCESS STATES:
────────────────
✅ Green success message box
• Shows after successful action
• Auto-disappears after 3 seconds


EMPTY STATES:
──────────────
📝 "Yorum yok" message
• Shows when no comments match filter
• Different messages per filter type
