📊 DATABASE STRUCTURE:

Table: comments
├── id (UUID) - Unique identifier
├── customer_name (text) - Customer's name [REQUIRED]
├── customer_email (text) - Customer's email
├── customer_image_url (text) - Profile image URL
├── comment_text (text) - The comment [REQUIRED]
├── rating (integer) - 1-5 stars
├── is_approved (boolean) - Published status
├── created_at (timestamp) - Creation date
└── updated_at (timestamp) - Last update date

============================================================

🔧 CUSTOMIZATION:

To change default rating:
  → Edit app/dashboard/comments/add/page.tsx, line with "rating: 5"

To change comment display limit on homepage:
  → Edit app/components/CustomerComments.tsx, add .slice(0, N)

To require approval before publishing:
  → Edit app/dashboard/comments/add/page.tsx, change is_approved default

To change default language from Turkish:
  → Search for Turkish labels and replace with your language
