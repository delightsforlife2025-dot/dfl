  1️⃣  Open: https://app.supabase.com
  
  2️⃣  Go to: SQL Editor → New Query
  
  3️⃣  Paste this SQL (copy below):
  
      ╔═══════════════════════════════════════════════════════════════════╗
      ║                                                                    ║
      ║ ALTER TABLE pages ENABLE ROW LEVEL SECURITY;                      ║
      ║ ALTER TABLE site_settings ENABLE ROW LEVEL SECURITY;              ║
      ║ ALTER TABLE nav_items ENABLE ROW LEVEL SECURITY;                  ║
      ║ ALTER TABLE social_links ENABLE ROW LEVEL SECURITY;               ║
      ║ ALTER TABLE contact_messages ENABLE ROW LEVEL SECURITY;           ║
      ║                                                                    ║
      ║ CREATE POLICY "Public pages" ON pages FOR SELECT                  ║
      ║   USING (is_published = true);                                    ║
      ║ CREATE POLICY "Public settings" ON site_settings FOR SELECT       ║
      ║   USING (true);                                                   ║
      ║ CREATE POLICY "Public nav" ON nav_items FOR SELECT                ║
      ║   USING (visible = true);                                         ║
      ║ CREATE POLICY "Public social" ON social_links FOR SELECT          ║
      ║   USING (true);                                                   ║
      ║ CREATE POLICY "Public contact" ON contact_messages FOR INSERT     ║
      ║   WITH CHECK (true);                                              ║
      ║                                                                    ║
      ╚═══════════════════════════════════════════════════════════════════╝
  
  4️⃣  Click: "Run" button
  
  5️⃣  Restart server: npm run dev
  
  6️⃣  Done! 🎉
