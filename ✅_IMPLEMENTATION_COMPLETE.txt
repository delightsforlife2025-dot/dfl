📞 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

PROBLEM: "Comments not showing on homepage"
SOLUTION: Check:
  1. Did you run the SQL? (most common!)
  2. Refresh the page
  3. Check browser console for errors
  4. Ensure is_approved = true in database

PROBLEM: "Admin page doesn't load"
SOLUTION: Check:
  1. You're logged in as admin
  2. You ran the SQL
  3. Refresh the browser
  4. Clear browser cache

PROBLEM: "Can't add comments"
SOLUTION: Check:
  1. You're on the add form page
  2. Fill all required fields
  3. Check browser console for errors
  4. Verify Supabase connection

PROBLEM: "Images not loading"
SOLUTION: Check:
  1. Image URL is valid
  2. Using HTTPS (not HTTP)
  3. Image still exists
  4. Try different image URL
